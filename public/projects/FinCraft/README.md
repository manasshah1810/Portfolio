# FinCraft

## Problem
Financial illiteracy remains a significant barrier to wealth creation, particularly among rural populations. Complex economic terms, stock market jargon, and the Fear Of Missing Out (FOMO) often lead to poor financial decisions or avoidance of investment opportunities altogether. Traditional educational methods are often too theoretical and fail to engage users in a practical way.

## Solution
FinCraft is a gamified financial education platform designed to make learning about money simple and accessible. The system provides an interactive environment where users can manage virtual cities, simulate stock market trades, and visualize the impact of long-term investments like SIPs and Mutual Funds. It removes the risk of real-world financial loss while providing deep insights into economic mechanics.

## Tech Stack
The project leverages a modern technology stack to ensure performance and scalability. The frontend is built with React Native and Expo, providing a seamless mobile experience. The intelligence layer is powered by Python and Scikit-learn, utilizing machine learning models to simulate realistic financial scenarios. Data handling and mathematical computations are optimized using NumPy, while future predictive features are designed with TensorFlow integration in mind.

## Key Idea
The core innovation of FinCraft is the Dynamic Financial Decision Bot. Unlike static educational tools, FinCraft incorporates a Machine Learning model trained on varied financial profiles. Using a Random Forest Classifier, the bot analyzes game parameters such as liquidity, debt-to-income ratio, and financial knowledge levels to make simulated investment decisions. This creates a realistic, adaptive learning environment that adjusts to the user's progress and difficulty level.

## Impact
The project has successfully simplified complex wealth-building concepts for users across different linguistic backgrounds. By providing a risk-free simulator, users can witness the power of compounding and the effects of market volatility firsthand. This practical experience builds confidence and empowers users to make better-informed financial decisions in their real lives.
