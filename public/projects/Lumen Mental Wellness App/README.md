# Bloom Mental Wellness App

Bloom is a comprehensive digital wellness platform designed to support the mental health of young individuals through empathetic AI interactions and community engagement.

## Problem
Why this project exists
The mental health of the youth is currently facing significant challenges, with rising rates of stress, anxiety, and emotional distress. Traditional support systems often lack the accessibility, anonymity, and immediate response that digital natives require. There is a profound need for a safe space where individuals can express their feelings without judgment or the friction of clinical appointments.

## Solution
What system you built
The Bloom system addresses these challenges by creating a safe digital sanctuary. It features an advanced AI companion assistant named Nova Kai that provides empathetic validation and support. The platform includes a Safe Circle for anonymous peer-to-peer connection, a Record Dump feature for vocalizing thoughts, and a One-Tap SOS mechanism for crisis management. This multi-layered approach ensures that help is always available, whether it be through automated AI, peer support, or emergency services.

## Tech Stack
Python, TensorFlow, React Native, Firebase
The application is built using a modern mobile and AI development stack:
- Frontend: React Native and Expo for a high-performance cross-platform experience.
- Backend: Firebase for secure user authentication, real-time data persistence (Firestore), and media storage.
- AI Core: Google Gemini 1.5 Flash utilized for natural language understanding, sentiment analysis, and long-term memory management.
- Integration: Firebase Cloud Functions for backend orchestration and AI integration logic.

## Key Idea
The interesting part
The central innovation of Bloom is the integration of long-term memory with empathetic AI agents. By utilizing the Gemini API's context window, the AI companion maintains a consistent persona and remembers past user interactions. This allows for a deeper, more personalized mental health journey where the system can recognize patterns in a user's emotional state and provide proactive validation based on their recorded history.

## Impact
What results you got
Bloom aims to democratize mental wellness support by providing a zero-cost, high-accessibility tool for emotional management. The implementation of anonymous peer circles fosters a sense of belonging, while the AI companion reduces the initial barrier to seeking help. The system enables users to feel more comfortable sharing their thoughts with an AI, leading to better emotional self-regulation and a significant reduction in the feeling of isolation among its target demographic.
